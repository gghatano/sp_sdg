要旨

本研究は、時系列データ拡張が分類精度に与える効果を UCR Time Series Classification Archive [{{ ref.ucr }}] の {{ facts.n_study_datasets }} データセット上で系統的に評価し({{ facts.n_study_runs }} run)、被験者ID付きデータ (UCI HAR [{{ ref.anguita2013 }}])へ展開して「所定の分類性能に必要な実被験者数をデータ拡張でどれだけ削減できるか」を定量する。 1D-CNN では {{ facts.n_significant }} 手法の拡張が精度を統計的に有意に改善する一方、MiniRocket [{{ ref.minirocket }}] では有意な改善が 見られず、効果はモデルに強く依存する。被験者数削減では、事前登録した目標精度 {{ facts.target_value if facts.target_value is not none else "—" }} に必要な実被験者数は拡張なしで約 {{ facts.n_star_none if facts.n_star_none is not none else "—" }} 名であり、点推定では DTW ベース合成 [{{ ref.forestier2017 }}] が最大 {{ facts.dtw_reduction_pct if facts.dtw_reduction_pct is not none else "—" }}% の削減を示す。しかしラベルを無作為化した negative control でも 同程度の見かけの削減が生じるため、本設定ではデータ拡張による被験者数削減を統計的に確定できない。 すべての結果は目標の事前登録とデータリーク防止プロトコルの下で得られ、実験基盤・レポートは完全に自動生成される。

1. 序論

ラベル付き信号データ、とりわけ被験者ごとに計測が必要な生体・行動データの収集は高コストである。データ拡張は学習データを 人工的に水増しして少データ性能を補う代表的手法だが、時系列領域では効果が手法・データ・モデルに依存することが報告されている [{{ ref.iwana2021 }}]。本研究はこの効果を公開データで追試したうえで、実応用上重要な問い——「拡張は実被験者の追加をどこまで代替できるか」——に答える。

研究課題を次の 2 点に定める。RQ1: 時系列拡張は分類精度をどの条件で改善するか(手法・モデル・学習データ量への依存)。 RQ2: 被験者ID付きデータにおいて、所定性能の達成に必要な実被験者数を拡張でどれだけ削減できるか。 仮説 H1(効果は低データ条件で大)、H2(効果はデータ・モデル依存で一様でない)、 H3(拡張は実被験者を部分的に代替する)を検証する。

本研究の貢献は以下のとおり。

  1. UCR {{ facts.n_study_datasets }} データセット × 7 拡張 × 2 モデル × 学習比率 5 段階での精度効果の系統評価と、多重比較補正付きの有意性検定。
  2. 被験者単位の学習曲線から必要被験者数 N* を推定し、削減率・等価被験者数・信頼区間を与える評価フレームワーク(目標性能は事前登録)。
  3. ラベル無作為化を用いた negative control による健全性検証。見かけの削減が信号のない対照でも再現されることを示し、削減主張に「対照越え」を課す。
  4. リーク防止・run manifest・監査・レポート生成まで完全自動化した再現可能な実験基盤。

2. 関連研究

時系列分類のデータ拡張については Iwana & Uchida のサーベイ [{{ ref.iwana2021 }}] が体系的評価を与え、効果がデータセット・モデルに依存し 一様でないことを報告している。ジッタリング・スケーリング等の摂動系はウェアラブル信号で有効性が示され [{{ ref.um2017 }}]、 mixup [{{ ref.mixup }}] は 2 サンプルの線形補間、SMOTE [{{ ref.smote }}] は近傍内挿による合成であり、いずれも本研究で評価する。 DTW を用いた系列合成は Forestier ら [{{ ref.forestier2017 }}] が提案した。分類器としては、時系列分類の標準ベンチマーク [{{ ref.bakeoff }}] で 高性能を示す ROCKET [{{ ref.rocket }}] とその後継 MiniRocket [{{ ref.minirocket }}]、および深層学習系の標準構成 [{{ ref.fawaz2019 }}] を用いる。 本研究はこれらの追試に加え、被験者数削減という応用軸での定量評価を独自に行う点が新しい。

3.4 評価プロトコルとリーク防止

  • データ拡張・スケーリング等の当てはめは学習データにのみ適用し、テストデータには一切触れない。
  • 被験者データでは同一被験者が train と test を跨がない group split を用い、コードで disjoint 性を検査する。
  • 目標性能は結果を見る前に事前登録する(§5、artifacts/pre_registration.md)。
  • 乱数 seed は run 単位で固定(分割・拡張・初期化に適用)し、全 run に一意 run_id と manifest を付す。

3.5 統計手法

拡張なし(none)との対比較は Wilcoxon 符号順位検定で行い、多重比較は Holm–Bonferroni 法(step-down)で補正して 有意水準 α=0.05 を判定する。被験者数削減の N* は学習曲線の線形補間で推定し、反復にわたる bootstrap で 95% 信頼区間を与える。

6. 考察

UCR での結果は、拡張効果がデータセット・モデル依存で一様でないという Iwana & Uchida [{{ ref.iwana2021 }}] の報告と整合する(H2 を支持)。 とくに MiniRocket ではどの拡張も有意でなく、生信号への摂動系拡張が変換ベース特徴に新情報を与えにくい可能性を示唆する。 ベースライン精度は既発表のベンチマーク相場 [{{ ref.bakeoff }}][{{ ref.minirocket }}] と一致し、実装の妥当性を支持する。 被験者数削減については、点推定では一部手法が削減を示すものの negative control と分離できず、H3(実被験者の代替)は本設定では確証されない。

8. 結論

時系列データ拡張の効果は手法・モデル・データに強く依存する。1D-CNN では複数手法が精度を有意に改善するが MiniRocket では改善せず、 被験者数削減に至っては negative control と区別できるレベルの効果は本設定では得られなかった。この結論は、削減を主張する際に 「対照越え」と反復数の確保が不可欠であることを示す。今後は反復数を増やした検出力設計、予備対象 WISDM [{{ ref.weiss2019 }}] での再現、 拡張強度と手法効果の分離が課題である。
