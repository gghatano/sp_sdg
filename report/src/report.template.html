被験者数削減評価(本研究の主目的)

{% if reduction and reduction.methods %}

UCI HAR(被験者ID付き)で、事前登録した目標性能({{ reduction.target_metric }} = {{ reduction.target_value }})に到達するのに必要な実被験者数 N* を、拡張なし(none)と各拡張手法で比較する。削減率 = 1 − N*(手法) / N*(none)。N* は学習曲線の線形補間で推定し、括弧内は 3 反復からの bootstrap 95% 信頼区間。目標達成に必要な被験者数の基準は N*(none) = {{ reduction.n_star_none }} 名{% if reduction.n_star_none_ci %}(95%CI {{ reduction.n_star_none_ci[0] }}–{{ reduction.n_star_none_ci[1] }}){% endif %}。

解釈上の注意: N* の信頼区間は反復 3 回のため広く、多くの手法で none の CI と重なる。したがって点推定では DTW・mixup 等が数%の削減を示すものの、3 反復では削減効果を統計的に確定できない。確度を上げるには反復数の増加が必要(限界セクション参照)。負の削減率は、その手法が必要被験者数をむしろ増やした(=有害)ことを意味する。

{% set nc = reduction.methods | selectattr('augmentation','equalto','label_shuffle') | first %} {% if nc and nc.reduction_rate is not none %}

Negative control(label_shuffle)の重要性

ラベルを無作為化した対照 label_shuffle(クラス信号を持たない)は {{ "%+.1f%%" | format(nc.reduction_rate * 100) }} の見かけの削減を示した。これは mixup・scaling の削減率と同等であり、約 4% 以下の削減は信号のない対照でも再現される=アーティファクトと区別できないことを意味する。少データではデータ量の水増し自体が正則化的に働くためと考えられる。したがって、削減の主張は「negative control を明確に超えるか」を条件とすべきで、本設定でそれを満たす手法はない(DTW の点推定のみが対照を上回るが信頼区間は重なる)。
